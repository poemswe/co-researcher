# Test Case: Thematic Analysis

## Metadata
- **Agent**: qualitative-research
- **Difficulty**: Medium
- **Focus**: Conducting thematic analysis on text data

## Rubric Profile
- **Primary**: qualitative-quality (75%)
- **Secondary**: output-structure (25%)
- **Timeout**: 900

## Task Prompt

```
Analyze these interview excerpts from employees about remote work and identify themes:

P1: "I love the flexibility. I can pick up my kids from school now. But sometimes I miss
the random chats by the coffee machine."

P2: "My productivity has skyrocketed. No more commute means I start fresh. But I do
feel isolated sometimes, especially during long projects."

P3: "The boundary between work and home has blurred. I find myself checking email at 10pm.
But I wouldn't go back - the autonomy is worth it."

P4: "Video calls are exhausting. Back-to-back Zoom meetings drain me more than in-person.
Yet overall, I have more control over my schedule."

P5: "I worry about my career progression. Out of sight, out of mind? But honestly,
I'm delivering better work than ever."

Conduct a thematic analysis and present the findings.
```

## Expected Behaviors

### Must Include
- [ ] At least 3 distinct themes identified
- [ ] Supporting quotes for each theme
- [ ] Theme definitions
- [ ] Pattern across participants noted

### Expected Themes
- Work-life balance/flexibility
- Social connection/isolation
- Productivity changes
- Autonomy vs. boundaries
- Career concerns

### Should Include
- [ ] Coding process described
- [ ] Subthemes where appropriate
- [ ] Contradictions/nuances noted
- [ ] Theme prevalence indicated
- [ ] Interpretation of meaning

### Should Not Include
- [ ] Themes without supporting evidence
- [ ] Missing obvious patterns
- [ ] Forcing data into preconceived categories
- [ ] Ignoring contradictory data

## Evaluation Criteria

### Reasoning Quality (Primary)
- Logical Coherence: Are themes well-grounded in data?
- Methodology Critique: Is the analysis systematic?

### Research Quality
- Comprehensiveness: Are key patterns captured?
- Accuracy: Are quotes accurately represented?

### Output Structure
- Clear theme presentation
- Supporting evidence
- Organized structure
