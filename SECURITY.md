# Security Policy

## Supported Versions

Security updates are provided for the latest code on the `main` branch.

## Reporting a Vulnerability

If you discover a security issue, please open a private security advisory in this repository:

- Go to **Security** → **Advisories** → **Report a vulnerability**
- Include reproduction steps, potential impact, and any suggested mitigation

If private advisory reporting is unavailable, open a GitHub issue and avoid disclosing exploit details publicly until a fix is available.
