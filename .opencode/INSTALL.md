# Installing Co-Researcher for OpenCode

## Prerequisites

- OpenCode.ai installed
- Node.js (for the plugin)

## Installation

1.  **Clone the repository** (if you hasn't already):
    ```bash
    git clone <repo-url> co-researcher
    cd co-researcher
    ```

2.  **Run the install script**:
    ```bash
    ./.opencode/install.sh
    ```

3.  **Restart OpenCode**.

## Verification

- Ask: "Do you have co-researcher powers?" (Should answer affirmatively).
- Run: `skill` (Should see `co-researcher` folder).
